# Local Mac TTS for Obsidian

Listen to your Obsidian notes using the **high-quality built-in macOS `say` engine**.

Use Premium voices like Ava, Daniel etc. to read your documents.

No internet • No API keys • No extra software required.

## Features

- Speak entire note 
- Pause / Resume playback
- Instant stop
- Save as **MP3** to Downloads folder
- Choose from all macOS voices
- Adjustable speech rate
- Ribbon icons + Command Palette

## Requirements

- **macOS only**
- Obsidian 0.15.0 or higher

## Installation

**Recommended:** Use [BRAT](https://github.com/TfTHacker/obsidian42-brat)  
Repo: `katnino/obsidian-mac-tts`

**Manual:** Put `main.js`, `manifest.json`, and `styles.css` in `.obsidian/plugins/mac-tts/`

## Usage

### Ribbon Icons

| Icon | Action |
|------|--------|
| 🔊   | Speak note |
| ⏸    | Pause / Resume |
| ⏹    | Stop |
| ⬇    | Save as MP3 |

## Settings
Choose voice, adjust speed, and test instantly.

## Saved Files
MP3 files are saved to `~/Downloads/` using the note name.

## Privacy
Completely local.
